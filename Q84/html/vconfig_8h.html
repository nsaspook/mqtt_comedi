Detailed Description

Software version and a brief doc for each version changes. Version for 47Q84.

V0.01 OPI daq slave via SPI2 V0.03 add all ADC channels and clean up for SECS defines and variables V0.04 start adding the DIO devices code with speed and mode changes 4MHz and mode 1, 50KHz mode 3 for the display V0.05 have the SPI I/O routines working for the DIO devices V0.10 mainly working DO version
